USAGE NOTES
================================================================================

These improvements are automatic - no code changes needed to use them.
Simply generate a map normally:

  generateFantasyWorld();
  fmRenderWorldMap();

All visual enhancements will be applied automatically based on:
  • Terrain heightmap and biome data
  • Water and moisture calculations
  • Kingdom and settlement locations
  • POI placements
  • Canvas dark mode setting

The improvements scale properly with map zoom levels and work in all
three rendering modes:
  • World map (full world view)
  • Kingdom map (zoomed kingdom view)
  • City map (zoomed city detail)
